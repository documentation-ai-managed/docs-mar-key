## Overview

Mar Key provides a centralized space to organize and manage your project documentation. You create custom spaces, structure content hierarchically, and collaborate seamlessly with your team. Access everything from a single dashboard at `https://dashboard.markey.com`.

<Callout kind="info">
Mar Key supports Markdown, MDX, and embeds for rich documentation.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://markey.com` and create your account with email or GitHub.
  </Step>
  <Step title="Create Space" icon="plus">
    Click **New Space** and name it for your project, like "Project Alpha Docs".
  </Step>
  <Step title="Add Content" icon="edit-3">
    Upload Markdown files or write directly. Use the editor for MDX components.
  </Step>
  <Step title="Share" icon="share-2">
    Generate a shareable link or invite team members via email.
  </Step>
</Steps>

## Use Cases

Tailor Mar Key to your team's needs.

<Tabs>
  <Tab title="Development Teams" icon="code">
    Document APIs, SDKs, and deployment guides. Embed live code examples from your repo.
    
    ```javascript
    // Example: Fetch project docs
    const response = await fetch('https://api.example.com/v1/docs?space=project-alpha');
    const docs = await response.json();
    console.log(docs);
    ```
  </Tab>
  <Tab title="Marketing Teams" icon="activity">
    Maintain product guides, FAQs, and launch checklists. Use rich embeds for videos and images.
  </Tab>
  <Tab title="Project Managers" icon="calendar">
    Centralize roadmaps, specs, and status updates. Track changes with version history.
  </Tab>
</Tabs>

<Expandable title="Advanced Integrations" default-open="false">
Integrate Mar Key with external tools using webhooks at `https://your-webhook-url.com/webhook`.

````javascript
// Webhook handler example
app.post('/webhook', (req, res) => {
  if (req.body.event === 'doc-updated') {
    // Sync to your CMS
    console.log('Doc updated:', req.body.data);
  }
  res.status(200).send('OK');
});
````
